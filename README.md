# YouTube Clone 🎬

A YouTube Clone built with ReactJS.

## Features:
- Home page with popular videos
- Video Player page
- Suggested videos
- Search functionality
- Login/Logout system
- Dark Mode Toggle 🌙
- Fully Responsive Design

## Live Demo
👉 [https://youtube-clone-react-olive.vercel.app/](https://youtube-clone-react-olive.vercel.app/)

## Built With:
- ReactJS
- Axios
- Pexels Video API
- Vercel Hosting
